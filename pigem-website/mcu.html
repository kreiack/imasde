Metamodelo Cartográfico Unificado (MCU)

El "lenguaje común" para datos geoespaciales de desastres: eliminando la fragmentación y habilitando la interoperabilidad automática

85% Auto-integración
95% Calidad Metadatos
99.5% Disponibilidad

El Problema: Heterogeneidad de Datos Geoespaciales

Las "islas tecnológicas" impiden la integración automática de información crítica

Sintáctica

Diferentes formatos de archivo

SHOA: Shapefile
INE: CSV
SERNAGEOMIN: KML
Requiere múltiples parsers

Esquemática

Diferentes estructuras de datos

"población" vs
"hab_total" vs
"num_habitantes"
Mapeo manual requerido

Semántica

Diferentes significados

"Zona riesgo alto" vs
"Amenaza nivel 3"
Interpretación ambigua

Temporal

Diferentes frecuencias

Censo: 10 años
Sísmica: tiempo real
Inconsistencias temporales

Fundamentos en Estándares Internacionales

ISO 19115/19139

Metadatos

Norma internacional para metadatos de información geográfica

Identificación: Título, resumen, palabras clave
Calidad: Linaje, precisión, completitud
Espacial: SRS, extensión geográfica
Temporal: Fecha creación, validez

WFS: Datos vectoriales
WCS: Datos de cobertura
CSW: Catálogo metadatos
WPS: Procesamiento remoto

Taxonomía de Capas de Información

AMENAZA

Sísmica: Fallas activas, zonificación SERNAGEOMIN
Tsunami: Cartas inundación, tiempos SHOA
Volcánica: Peligros, lahares SERNAGEOMIN
Incendios: Índice peligro, combustible CONAF

VULNERABILIDAD

Física: Tipología, año construcción INE, Municipios
Social: Educación, dependencia INE, CASEN
Económica: Avalúos, actividad SII, INE
Ambiental: Ecosistemas frágiles MMA

CAPACIDAD

Infraestructura: Hospitales, escuelas Municipios
Transporte: Red vial, puentes MOP
Comunicaciones: Antenas, fibra SUBTEL
Respuesta: Albergues, equipos SENAPRED
